## ðŸ“¦ QUICK START

### 1. Server Setup (Windows)
- Go to `v2.6.0-Final` folder.
- Run `SysTracker_Server.exe`.
- Open `http://localhost:7777` in your browser.

### 2. Agent Setup
- Once logged into the dashboard, go to **Settings** -> **Deployment & Downloads**.
- Download the **SysTracker_Agent.exe**.
- Run it on any machine you wish to monitor.
