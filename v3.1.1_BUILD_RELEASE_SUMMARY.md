## üîß How to Rebuild Components

### Prerequisites
- Node.js 18+ (for build tools)
- npm 11.6+ (for package management)
- Python 3.8+ (optional, for agent development)
- PowerShell 5.0+ (for Windows scripts)
- Windows SDK (optional, for MSIX packaging)

### Option 1: Quick Rebuild (Recommended)

**Rebuild Everything:**
```bash
# Navigate to project root
cd /workspaces/SysTracker

# Update version numbers if needed
npm version patch  # or minor/major

# Rebuild dashboard
cd dashboard && npm run build && cd ..

# The server doesn't need special build steps
# Agent executable is pre-built

# Optional: Rebuild MSIX package (Windows only)
# powershell .\build_msix.ps1
```

### Option 2: Component-by-Component Rebuild

**Dashboard:**
```bash
cd dashboard
npm install
npm run build
# Output: out/ directory with static HTML/JS/CSS
```

**Server:**
```bash
cd server
npm install
# No build needed - uses source files directly
npm test  # Run tests
```

**Agent Executable:**
```bash
cd agent
# If using Windows, compile with PyInstaller:
pip install -r requirements.txt
pyinstaller SysTracker_Agent.spec
# Output: dist/SysTracker_Agent.exe
```

**MSIX Package:**
```powershell
# Windows PowerShell (Administrator)
# Ensure agent/SysTracker_Agent.exe exists first
.\build_msix.ps1
# Output: SysTracker_Agent.msix
```

### Option 3: Custom Builds

**Rebuild with Custom Version:**
```bash
# Update package.json files with new version
# Then:
cd dashboard && npm run build
```

**Rebuild Agent for Different Platform:**
```bash
cd agent
# Edit SysTracker_Agent.spec to change target_arch
# Then regenerate with PyInstaller
```

## üìû Support & Troubleshooting

### If Installation Has Issues

1. **Run Diagnostic Tool:**
   ```powershell
   .\agent\FIND_APP_INSTALLATION.ps1
   ```

2. **Check Logs:**
   - Windows Event Viewer ‚Üí Application
   - `C:\Program Files\SysTracker Agent\logs\`

3. **Review Guide:**
   - See `AGENT_INSTALLATION_TROUBLESHOOTING.md`

### Common Issues

| Issue | Solution |
|-------|----------|
| "Path not found" | Run diagnostic script to find actual location |
| "Access denied" | Run installer as Administrator |
| "App not found after install" | Check `C:\Program Files\SysTracker Agent\` |
| "Multiple installations" | Use diagnostic script, uninstall extras |
